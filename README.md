# Library Management System

This is a web-based Library Management System developed using Python and Django.

## Features
- Add new books
- Update book details
- Delete books
- View book list
- Simple and user-friendly interface

## Technologies Used
- Python
- Django
- MySQL
- HTML
- CSS

## How to Run the Project
1. Clone the repository
2. Install required packages
3. Run migrations
4. Start the Django server

## Purpose
This project was created to practice Django concepts such as models, views, templates, and database operations.
